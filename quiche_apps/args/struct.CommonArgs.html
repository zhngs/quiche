impl Args for CommonArgs

Creates a new CommonArgs structure using the provided Docopt.

The Docopt usage String needs to include the following:

–http-version VERSION HTTP version to use. –max-data BYTES Connection-wide flow control limit. –max-window BYTES Connection-wide max receiver window. –max-stream-data BYTES Per-stream flow control limit. –max-stream-window BYTES Per-stream max receiver window. –max-streams-bidi STREAMS Number of allowed concurrent streams. –max-streams-uni STREAMS Number of allowed concurrent streams. –dump-packets PATH Dump the incoming packets in PATH. –no-grease Don’t send GREASE. –cc-algorithm NAME Set a congestion control algorithm. –disable-hystart Disable HyStart++. –dgram-proto PROTO DATAGRAM application protocol. –dgram-count COUNT Number of DATAGRAMs to send. –dgram-data DATA DATAGRAM data to send.
